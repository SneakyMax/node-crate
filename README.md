node-crate
==========

Node.js base DB-Driver for CRATE (www.crate.io)

## Installation

## Usage

```js
var crate = require('node-crate.js');
```

```js
crate.execute
```

```js
crate.insert
```

```js
crate.update
```

```js
crate.delete
```

#### Blobs

```js
crate.insertBlob
```

```js
crate.insertBlobFile
```

```js
crate.getBlob
```
